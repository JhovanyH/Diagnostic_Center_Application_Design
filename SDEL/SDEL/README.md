HOW TO RUN:
1. Make sure Python is installed (python.org)
2. Open terminal/PowerShell in this folder
3. Install requirements:
      pip install customtkinter pillow
4. Run the app:
      python main.py

The database (purehealthDb.db) is included with sample data.
No additional setup needed.
